1 Objetivo General: Lograr que los estudiantes comprendan y generalicen las nuevas tendencias básicas del Diseño de Sitio Web que representa una estructura de forma correctas en los Lenguajes HTML5 y CSS, para poder analizar y sobre todo desarrollar aplicaciones web siguiendo estas tendencias.

2 Objetivo Especificos: Reconocer e Identificar las nuevas características para que los estudiantes pueda operar y administrar sistemas de control de versiones como es HTML de la WWW (Word Wide Web). Aprender El uso Correcto los lenguajes HTML5 y CSS3 para aplicarlos en la solución de problemas de construcción de Sitios Web.

3Desarrollar una página para que los estudiantes pueda evaluar y desarrollar Sitios Web que satisfagan las necesidades del usuario y dar recomendaciones concretas para su actualización.
